To use the animation, first load an automata from the database. This should be displayed graphically. The automata can be zoomed in and out with the use of the '+' and '-' keys as desired.

To load inputs to the automata, you can either load a CSV file of comma separated strings, by pressing the load batch button. Otherwise, individual strings can be added to the queue by using the add string button. Right click items on the queue to delete them.

Press play to send the automata forward with one transition. Press the fastforward button to allow it to run as an animation. This animation can be sped-up or slowed down with the slider bar.
