To create a new automata, simply select the PDA designer tab and then start building it.

To place a node, select the node entry, and click on the drawing board. A node details panel will open up and the details of the individual node can be edited (including whether any self-looping relations loop down or up for aesthetic purposes).

Create transitions by selecting the line button. Then drag a line between two nodes, and enter in whatever transitions you want to define between them.

Use the selector tool to edit existing nodes by clicking on them or dragging their position. Relations can have their details changed.

The canvas can be expanded to allow more room for drawing. When finished click save and you will be directed to place the automata in a directory which exists within the database.
